# Campaign Engine

Campaign Engine is a self-hosted, extensible campaign management platform for tabletop RPGs.

Unlike traditional campaign managers that are tightly coupled to a single ruleset, Campaign Engine separates storytelling from game mechanics.

The Core Engine manages:

- Campaigns
- Users
- Sessions
- Events
- Characters
- Organizations
- Locations
- Relationships
- Notes
- Media
- Permissions

Game systems are implemented as plugins.

Examples:

- Vampire: The Masquerade 5th Edition
- Call of Cthulhu 7e
- D&D 5e
- Pathfinder 2e

## Philosophy

The engine manages stories.

Plugins manage rules.

No game-specific logic belongs in the Core.
